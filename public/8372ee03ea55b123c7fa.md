## はじめに
生成AI等の検証を簡単に容易でいる環境が欲しいと思い、「Amazon SageMaker Studio コードエディタ」を利用する方法があったので、環境構築のための手順を公開します。

##### Amazon SageMaker Studio の Code Editor 機能とは？
**ざっくり概要**
 - VS Code オープンソースを基盤とするエディタ
 - ユーザのコードの記述、テスト、デバッグ、実行を支援
 - Amazon SageMaker Studio と完全統合されている

AWS:
[Amazon SageMaker Studio のコードエディタ](https://docs.aws.amazon.com/ja_jp/sagemaker/latest/dg/code-editor.html)

## 手順
### 1.SageMakerドメインの作成
ドメイン作成に関しては金額は発生しません

#### 1-1.リソース検索にて「Amazon SageMaker AI」を検索・選択
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/7cc1957a-c20f-4f73-bf1e-0888f869647e.png)


#### 1-2.左ペインより「ドメイン」押下
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/354525a1-51b8-48fd-ba76-e9e7ac0ebb00.png)


#### 1-3.デフォルトで選択されている「シングルユーザ向けの設定(クイックセットアップ)」で「設定」押下
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/a02eab69-074d-446c-bc58-e773760e01a3.png)


#### 1-4.「SageMaker Domain の準備ができました」の表示が出るまで待機
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/784b41d0-9174-4d3e-bd46-ba5ae3d8b467.png)


### 2.IAMによる権限
#### 2-1.「1-4」設定後、タブ「ドメインの設定」にある「認証と許可」でロール確認
以下ロール名をIAMより検索し、必要ポリシーのアタッチを行う
 - AmazonSageMaker-ExecutionRole-<15桁の文字列>
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/bd305a4a-fa6f-45e5-b3c9-c83090ead684.png)


#### 2-2.リソース検索にて「IAM」を検索・選択
赤枠部分を選択
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/f04c76b7-3a0b-4fb3-abb3-e9feeef2fb10.png)


#### 2-3.IAMロールの検索
「2-1」で確認したロール名で検索
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/7257c7d9-c26c-429e-9935-14d8cc2ddf45.png)


#### 2-4.デフォルトポリシーの状態確認
利用用途に応じて、ポリシーのアタッチ・デタッチを行う
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/2e3dc5ad-0a3c-46ad-b0cb-d4a25f3ef289.png)


### 3.SageMaker Studio Code Editor(プライベートスペース)の作成
#### 3-1.タブ「ユーザープロファイル」タブを選択、「起動」押下しプルダウンより「Studio」を選択し、「SageMaker Studio」の画面へ遷移
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/2019e6df-f957-44ed-946d-d6fa3045c34d.png)


#### 3-2.左上「Code Eitor」押下
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/de679225-1afe-4dee-940e-c12e85aabb27.png)


#### 3-3.「Create Code Editor space」押下
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/c2bac155-7016-479a-8236-ae7505628cbc.png)


#### 3-4.「Name」に任意の名前を入力し「Create space」押下
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/852a2524-6d5c-4aa2-bd61-2ea80806725f.png)


### 4.インスタンス起動
#### 4-1.「Run space」を押下
※コードエディタ用 インスタンスのスペックに応じて利用料金が発生
[Amazon SageMaker の料金](https://aws.amazon.com/jp/sagemaker/pricing/)

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/1c8e6180-54b3-437b-98b5-56dec5a67939.png)


#### 4-2.「Open Code Editor」を押下して、コードエディタ画面へ遷移
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/b66b7aaa-689d-40ba-aa75-446d1285addc.png)


### 5.コードエディタの画面操作
#### 5-1.遷移後の画面
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/575f1c35-b4e1-45a4-9be5-3398897949c9.png)


#### 5-2.初期設定
左ペイン「ファイルマーク」クリック、「Open Folder」を選択して、画面に表示される「Open Folder」横の「OK」を押下
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/8ce4298a-22dd-430f-b522-664f5a03e117.png)


#### 5-3.ポップアップ「フォルダ内のファイルの作成者を信頼しますか？」の表示
「Trust the authors（以下略）」にチェックし、「Yes, I trust the authors」を押下
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/ad306bcb-c7b3-4fbe-864f-c19d31a1786b.png)


#### 5-4.ターミナル画面の表示の方法
画面右上(赤枠部分)を押下することでターミナル画面が表示される
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/694365/0e862ea1-adfd-46aa-a85f-0d3ad348637f.png)


### 6.停止方法
#### 6-1.SageMaker Studio画面より「Stop」を押下
