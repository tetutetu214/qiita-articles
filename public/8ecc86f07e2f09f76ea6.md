## 1.はじめに
### 1.1.構築背景
Google Cloud を仕事で利用し始めて 8ヶ月。そろそろ、ちゃんとキャッチアップしようと思いました。
前回構築した[AWSの最新情報を取得・翻訳して毎朝 自動通知 ハンズオン](https://qiita.com/tetutetu214/items/333b75534f64ee3b1bd6)を、折角Google Cloud のリソースの最新ニュースを取得するので、Google Clouのリソースに置き換えて構築していきたいと思います。

出力先も日々の利用がしやすいように LINE 通知するようにします。

```shell
import json
import uuid
import time
import re
from datetime import datetime, timezone, timedelta
from typing import List, Dict
from html import unescape
import functions_framework
import feedparser
import requests
from google.cloud import firestore


def extract_product_updates(html_content: str, entry_title: str, entry_link: str, entry_updated: str) -> List[Dict]:
    """
    HTMLコンテンツから製品別の更新情報を抽出する
    
    Args:
        html_content: HTMLコンテンツ
        entry_title: エントリのタイトル（日付）
        entry_link: エントリのリンク
        entry_updated: エントリの更新日時
        
    Returns:
        製品別更新情報のリスト
    """
    if not html_content:
        return []
    
    # HTMLエンティティをデコード
    content = unescape(html_content)
    
    # 製品別更新情報リスト
    product_updates = []
    
    # h2タグ（製品タイトル）とその後のコンテンツを抽出
    product_pattern = r'<h2[^>]*class="release-note-product-title"[^>]*>([^<]+)</h2>(.*?)(?=<h2[^>]*class="release-note-product-title"|$)'
    product_matches = re.findall(product_pattern, content, re.DOTALL | re.IGNORECASE)
    
    for product_title, product_content in product_matches:
        product_title = product_title.strip()
        
        # h3タグ（変更種別）とその内容を抽出
        section_pattern = r'<h3[^>]*>([^<]+)</h3>(.*?)(?=<h3[^>]*>|$)'
        section_matches = re.findall(section_pattern, product_content, re.DOTALL | re.IGNORECASE)
        
        sections = []
        if section_matches:
            for section_title, section_content in section_matches:
                # HTMLタグを除去してテキストのみ抽出
                clean_content = re.sub(r'<[^>]+>', '', section_content)
                clean_content = re.sub(r'\s+', ' ', clean_content).strip()
                
                if clean_content:  # 内容がある場合のみ追加
                    sections.append({
                        'type': section_title.strip(),
                        'content': clean_content
                    })
        else:
            # h3がない場合は全体のコンテンツを抽出
            clean_content = re.sub(r'<[^>]+>', '', product_content)
            clean_content = re.sub(r'\s+', ' ', clean_content).strip()
            
            if clean_content:
                sections.append({
                    'type': 'Update',
                    'content': clean_content
                })
        
        # 製品別の更新情報を作成
        if sections:
            product_updates.append({
                'product': product_title,
                'date': entry_title,
                'sections': sections,
                'link': entry_link,
                'updated': entry_updated
            })
    
    return product_updates


def fetch_rss_feed(url: str) -> dict:
    """
    RSSフィードを取得して解析する
    
    Args:
        url: RSSフィードのURL
        
    Returns:
        解析されたフィードデータ
    """
    try:
        response = requests.get(url, timeout=30)
        response.raise_for_status()
        return feedparser.parse(response.content)
    except Exception as e:
        print(f"RSS取得エラー: {e}")
        raise


def filter_recent_entries(entries: list, hours_back: int = 24) -> list:
    """
    指定時間前までのエントリをフィルタリング
    
    Args:
        entries: 全エントリのリスト
        hours_back: 何時間前までのエントリを取得するか
        
    Returns:
        指定時間前までのエントリのリスト
    """
    # JST設定
    jst = timezone(timedelta(hours=9))
    now = datetime.now(jst)
    
    # 過去X時間分の日付リストを作成
    target_dates = []
    for i in range(hours_back // 24 + 1):  # 日数分ループ
        target_date = (now - timedelta(days=i)).strftime("%B %d, %Y")
        target_dates.append(target_date)
    
    print(f"対象日付（過去{hours_back}時間）: {target_dates}")
    
    recent_entries = []
    for entry in entries:
        entry_title = entry.get('title', '')
        print(f"エントリタイトル: {entry_title}")
        
        # タイトルに対象日付が含まれているかチェック
        for target_date in target_dates:
            if target_date in entry_title:
                recent_entries.append(entry)
                print(f"対象エントリとして追加: {entry_title}")
                break
    
    print(f"対象エントリ数: {len(recent_entries)}")
    return recent_entries


def save_to_firestore(entries: list) -> int:
    """
    エントリをFirestoreに保存する（製品別に分割）
    
    Args:
        entries: 保存するエントリのリスト
        
    Returns:
        保存した件数
    """
    db = firestore.Client()
    collection_ref = db.collection('rss_entries')
    
    saved_count = 0
    current_time = int(time.time())
    # TTL: 7日後に自動削除
    ttl_time = current_time + (7 * 24 * 60 * 60)
    
    for entry in entries:
        # HTMLコンテンツから製品別更新情報を抽出
        html_content = entry.get('content', [{}])[0].get('value', '') if entry.get('content') else entry.get('summary', '')
        product_updates = extract_product_updates(
            html_content, 
            entry.get('title', ''),
            entry.get('link', ''),
            entry.get('updated', '')
        )
        
        # 製品ごとに別々のドキュメントとして保存
        for product_update in product_updates:
            for section in product_update['sections']:
                # ドキュメントIDを生成
                doc_id = str(uuid.uuid4())
                
                # ドキュメントデータ作成
                doc_data = {
                    'id': doc_id,
                    'title': f"{product_update['date']} - {product_update['product']}",
                    'product': product_update['product'],
                    'date': product_update['date'],
                    'type': section['type'],
                    'content': section['content'],
                    'link': product_update['link'],
                    'updated': product_update['updated'],
                    'timestamp': current_time,
                    'ttl': ttl_time
                }
                
                # 重複チェック（製品名、日付、タイプで判定）
                existing_docs = collection_ref.where('product', '==', doc_data['product']).where('date', '==', doc_data['date']).where('type', '==', doc_data['type']).limit(1).get()
                
                if not existing_docs:
                    collection_ref.document(doc_id).set(doc_data)
                    saved_count += 1
                    print(f"保存完了: {doc_data['title']} - {doc_data['type']}")
                else:
                    print(f"重複スキップ: {doc_data['title']} - {doc_data['type']}")
    
    return saved_count


@functions_framework.http
def rss_fetch_handler(request):
    """
    HTTP トリガーでRSS取得処理を実行（過去24時間分）
    """
    try:
        # Google Cloud RSSフィードURL
        rss_url = "https://cloud.google.com/feeds/gcp-release-notes.xml"
        
        print(f"RSS取得開始: {rss_url}")
        
        # RSSフィード取得
        feed = fetch_rss_feed(rss_url)
        
        if not feed.entries:
            return {'status': 'error', 'message': 'RSS エントリが見つかりません'}, 400
        
        # 過去24時間分のエントリをフィルタリング
        recent_entries = filter_recent_entries(feed.entries, hours_back=24)
        
        if not recent_entries:
            result = {
                'status': 'success',
                'message': '過去24時間の更新情報はありませんでした',
                'total_entries': len(feed.entries),
                'saved_count': 0
            }
            print(f"処理完了: {result}")
            return result, 200
        
        # Firestoreに保存
        saved_count = save_to_firestore(recent_entries)
        
        result = {
            'status': 'success',
            'message': f'過去24時間分RSS取得完了: {saved_count}件保存',
            'total_entries': len(feed.entries),
            'recent_entries': len(recent_entries),
            'saved_count': saved_count
        }
        
        print(f"処理完了: {result}")
        return result, 200
        
    except Exception as e:
        error_msg = f"RSS取得処理エラー: {str(e)}"
        print(error_msg)
        return {'status': 'error', 'message': error_msg}, 500
EOF
```

```shell
import json
import os
import time
from datetime import datetime, timezone, timedelta
from typing import List, Dict
import functions_framework
import requests
from google.cloud import firestore


def get_recent_entries() -> List[Dict]:
    """
    最近のエントリをFirestoreから取得(30分以内)
            
    Returns:
        エントリのリスト
    """
    db = firestore.Client()
    collection_ref = db.collection('rss_entries')
    
    # 指定時間前のタイムスタンプ
    cutoff_time = int(time.time()) - (30 * 60)
    
    # 最近のエントリを取得
    docs = collection_ref.where('timestamp', '>', cutoff_time).order_by('timestamp', direction=firestore.Query.DESCENDING).get()
    
    entries = []
    for doc in docs:
        entries.append(doc.to_dict())
    
    return entries


def format_summary(entries: List[Dict]) -> str:
    """
    エントリを日付別にグループ化してフォーマット
    
    Args:
        entries: エントリのリスト
        
    Returns:
        フォーマットされたテキスト
    """
    if not entries:
        return "新しい更新はありません。"
    
    # 日付別にグループ化
    entries_by_date = {}
    for entry in entries:
        date = entry.get('date', '')
        if date not in entries_by_date:
            entries_by_date[date] = []
        entries_by_date[date].append(entry)
    
    # 日付順でソート
    sorted_dates = sorted(entries_by_date.keys())
    
    # フォーマット作成
    summary_text = ""
    
    for date in sorted_dates:
        date_entries = entries_by_date[date]
        summary_text += f"Google Cloud更新情報 ({date}) - {len(date_entries)}件の更新\n\n"
        
        for i, entry in enumerate(date_entries, 1):
            product = entry.get('product', '')
            update_type = entry.get('type', '')
            content = entry.get('content', '')
            link = entry.get('link', '')
            
            # タイトル作成
            title = f"{product} - {update_type}"
            
            # 内容を200文字に制限
            short_content = content[:200] + "..." if len(content) > 200 else content
            
            summary_text += f"【{i}】Title: {title}\n"
            summary_text += f"内容: {short_content}\n"
            summary_text += f"URL: {link}\n\n"
    
    return summary_text


def send_line_message(message: str) -> bool:
    """
    LINE Messaging APIでメッセージ送信
    
    Args:
        message: 送信するメッセージ
        
    Returns:
        送信成功フラグ
    """
    # 環境変数から設定取得
    channel_access_token = os.environ.get('LINE_CHANNEL_ACCESS_TOKEN')
    user_id = os.environ.get('LINE_USER_ID')
    
    if not channel_access_token or not user_id:
        print("LINE API設定が不足しています")
        return False
    
    url = "https://api.line.me/v2/bot/message/push"
    headers = {
        "Authorization": f"Bearer {channel_access_token}",
        "Content-Type": "application/json"
    }
    
    # LINEメッセージの文字数制限対応（5000文字まで）
    if len(message) > 5000:
        message = message[:4990] + "\n...(続く)"
    
    data = {
        "to": user_id,
        "messages": [{
            "type": "text",
            "text": message
        }]
    }
    
    try:
        response = requests.post(url, headers=headers, json=data, timeout=30)
        response.raise_for_status()
        print("LINE送信成功")
        return True
    except Exception as e:
        print(f"LINE送信エラー: {e}")
        return False


@functions_framework.http
def summary_send_handler(request):
    """
    HTTP トリガーで要約・送信処理を実行
    """
    try:
        print("要約・送信処理開始")
        
        # 現在時刻（日本時間）
        jst = timezone(timedelta(hours=9))
        current_time = datetime.now(jst).strftime("%Y/%m/%d %H:%M")
        
        # 最近のエントリ取得
        entries = get_recent_entries()
        print(f"取得エントリ数: {len(entries)}")
        
        if not entries:
            message = f"Google Cloud更新情報 ({current_time})\n\n新しい更新はありません。"
        else:
            # エントリをフォーマット
            summary = format_summary(entries)
            message = f"{summary}"
        
        # LINE送信
        success = send_line_message(message)
        
        if success:
            result = {
                'status': 'success',
                'message': f'要約・送信完了: {len(entries)}件処理',
                'entries_count': len(entries)
            }
        else:
            result = {
                'status': 'error',
                'message': 'LINE送信に失敗しました'
            }
        
        print(f"処理完了: {result}")
        return result, 200 if success else 500
        
    except Exception as e:
        error_msg = f"要約・送信処理エラー: {str(e)}"
        print(error_msg)
        return {'status': 'error', 'message': error_msg}, 500
EOF
```

### 2.5. Workflows作成
2つのFunctionを順次実行するワークフローを作成する。

|項番|フロー名|
|---|---|
|1|RSS取得 Function実行|
|2|要約送信 Function実行|
|3|エラーハンドリング|

<details>
<summary>1.ワークフロー定義ファイル作成</summary>

```shell
# Function URL取得
export RSS_FUNCTION_URL=$(gcloud functions describe rss-fetch-function \
  --region=$REGION \
  --project=$PROJECT_ID \
  --format="value(url)")

export SUMMARY_FUNCTION_URL=$(gcloud functions describe summary-send-function \
  --region=$REGION \
  --project=$PROJECT_ID \
  --format="value(url)")

# ワークフローディレクトリ移動
cd workflows

# ワークフローのyamlファイル作成
cat > notification-workflow.yaml << EOF
main:
  steps:
    - fetch_rss:
        call: http.post
        args:
          url: "$RSS_FUNCTION_URL"
        result: fetch_result
    
    - log_fetch_result:
        call: sys.log
        args:
          data: \${fetch_result}
          severity: "INFO"
    
    - send_summary:
        call: http.post
        args:
          url: "$SUMMARY_FUNCTION_URL"
        result: send_result
    
    - log_send_result:
        call: sys.log
        args:
          data: \${send_result}
          severity: "INFO"
    
    - return_result:
        return: \${send_result}
EOF

# 元のディレクトリに戻る
cd ..
```
</details>

<details>
<summary>2.ワークフローデプロイ</summary>

```shell
# ワークフローデプロイ
gcloud workflows deploy notification-workflow \
  --source=workflows/notification-workflow.yaml \
  --location=$REGION
```
</details>

<details>
<summary>3.ワークフロー実行テスト</summary>


### 2.6. Cloud Scheduler作成
毎朝 6時(JST)に自動実行のスケジューラを作成する。

<details>
<summary>1.IAMロール作成</summary>

```shell
# プロジェクト番号取得
export PROJECT_NUMBER=$(gcloud projects describe $PROJECT_ID --format="value(projectNumber)")

# Cloud SchedulerのサービスアカウントにWorkflows実行権限を付与
gcloud projects add-iam-policy-binding $PROJECT_ID \
  --member="serviceAccount:service-$PROJECT_NUMBER@gcp-sa-cloudscheduler.iam.gserviceaccount.com" \
  --role="roles/workflows.invoker"
```
</details>

<details>
<summary>2.スケジューラ作成</summary>

```shell
# Cloud Schedulerジョブ作成
gcloud scheduler jobs create http rss-notification-job \
  --schedule="0 6 * * *" \
  --uri="https://workflowexecutions.googleapis.com/v1/projects/$PROJECT_ID/locations/$REGION/workflows/notification-workflow/executions" \
  --http-method=POST \
  --time-zone="Asia/Tokyo" \
  --location=$REGION \
  --oauth-service-account-email="$PROJECT_NUMBER-compute@developer.gserviceaccount.com"

# 作成確認
gcloud scheduler jobs describe rss-notification-job --location=$REGION
```
</details>
